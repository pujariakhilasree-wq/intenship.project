This repository contains projects developed during an Artificial Intelligence Internship at CodeSoft. The focus is on building intelligent systems and simple AI-based applications.
📌 Projects Included
🎯 1. Recommendation System (Predictive System)
An AI-based system that predicts user preferences.
Recommends items (such as movies/products) based on similarity.
Uses content-based filtering and intelligent logic.
💬 2. Chatbot System
An AI chatbot designed for user interaction.
Simulates human conversation.
Responds to user queries using basic NLP concepts.
❌⭕ 3. Tic Tac Toe (Game AI)
A classic Tic Tac Toe game with AI logic.
The system can play against a human user.
Demonstrates decision-making and game strategy.
🛠️ Technologies Used
Python 🐍
Pandas
NumPy
Basic AI & NLP Concepts
